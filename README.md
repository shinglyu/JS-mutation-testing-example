JavaScript Mutation Testing Example
===========

Using [grunt-mutation-testing](https://www.npmjs.com/package/grunt-mutation-testing)

# Installation
* `git clone` this repo
* 

```
sudo npm install -g mocha
sudo npm install -g grunt-cli
npm install .
```

# Running

* Try a passing test:

```
mocha
```

* Run the mutation testing:

```
grunt
```
